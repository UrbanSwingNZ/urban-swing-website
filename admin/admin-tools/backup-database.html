About This Tool

This tool creates a complete backup of your Firestore database by exporting collections to a ZIP file.

  • Select which collections to backup
  • Creates a single ZIP file containing all selected collections as JSON files
  • ZIP file is timestamped for easy version tracking (e.g., backup-2025-10-22-15-30-00.zip)
  • Backups can be extracted and used for restoration or migration
